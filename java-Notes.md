#  Java Notes

##  Introduction
Java is an object-oriented programming language developed by Sun Microsystems.

##  Key Features
- Platform Independent (Write Once, Run Anywhere)
- Object Oriented
- Robust & Secure
- Multithreaded

##  Example Program
```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
